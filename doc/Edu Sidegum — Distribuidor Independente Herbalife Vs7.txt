### ### 1. CAPÍTULO I — DIRETRIZES ESTRATÉGICAS

#### #### 1.1. Propósito do Ecossistema
O ecossistema HLF não é um conjunto de landing pages soltas. É uma **rede semântica de autoridade regional** que demonstra ao Google E-E-A-T (Experiência, Especialidade, Autoridade e Confiança) por meio de conteúdo estruturado, interligação contextual e identificação clara do Distribuidor Independente.

Cada página, componente e ativo existe para:
- Atrair tráfego local qualificado em **Novo Hamburgo/RS**;
- Educar o lead sobre nutrição de alta performance;
- Conduzir o prospecto por um funil consultivo (**SPIN Selling**);
- Converter em Cliente Premium ou Novo Consultor;
- Fortalecer a autoridade regional do consultor.

A identidade do projeto baseia-se na figura do consultor como autoridade técnica e ponto de confiança para o consumidor final.

#### #### 2.4. Sistema de Injeção de Componentes (bloco-incluir.js)
O sistema permite que componentes HTML compartilhados sejam mantidos em UM ÚNICO local e injetados dinamicamente.

**Código Fonte:**
```javascript
function injetar(arquivo, seletorAlvo, posicao) {
    return fetch(BASE + arquivo + '?v=' + Date.now())
    .then(r => r.text())
    .then(html => {
        const alvo = document.querySelector(seletorAlvo);
        if (alvo) alvo.insertAdjacentHTML(posicao, html);
    })
    .catch(() => {});
}
injetar('compartilhado/bloco-cookies.html', 'body', 'beforeend');
```

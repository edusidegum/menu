## 5. CHECKLIST DE SEO ON-PAGE (Toda Página)

```html
<!-- ✅ HEAD COMPLETO -->
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>[H1 da página] — [Marca]</title>         <!-- 50-60 caracteres -->
<meta name="description" content="[150-160 caracteres com palavra-chave]">
<meta name="keywords" content="[5-8 termos]">
<meta name="author" content="edusidegum W">
<meta name="robots" content="index, follow">
<link rel="canonical" href="[URL absoluta]">

<!-- ✅ OPEN GRAPH -->
<meta property="og:title" content="[igual ao title]">
<meta property="og:description" content="[mesma description]">
<meta property="og:url" content="[URL absoluta]">
<meta property="og:image" content="[URL absoluta da imagem 1200x630]">
<meta property="og:image:width" content="1200">
<meta property="og:image:height" content="630">
<meta property="og:type" content="website">
<meta property="og:site_name" content="[Marca]">
<meta property="og:locale" content="pt_BR">

<!-- ✅ GOOGLE TAG MANAGER (placeholder) -->
<!-- Google Tag Manager -->
<!-- SUBSTITUIR_GTM -->
```

```html
<!-- ✅ HIERARQUIA HTML -->
<h1 class="titulo-principal">[Título único da página]</h1>
<h2>[Seção 1]</h2>
<h2>[Seção 2]</h2>
<h3>[Sub-seção, se necessário]</h3>

<!-- ✅ IMAGENS -->
<img src="img/[arquivo].webp" alt="[descrição com palavra-chave]" loading="lazy" width="[w]" height="[h]">

<!-- ✅ LINKS INTERNOS -->
<a href="[pagina].html">[âncora descritiva]</a>
```

## 9. COMANDOS ÚTEIS NO TERMINAL (Ubuntu)

```bash
# Criar nova página a partir do modelo
cp em_obras.html nova-pagina.html

# Buscar e substituir em lote
sed -i 's/SUBSTITUIR_TITULO/Meu Título/g' nova-pagina.html

# Validar HTML
tidy -q -e nova-pagina.html

# Comprimir imagens para WebP
for f in img/*.png; do cwebp "$f" -o "${f%.png}.webp"; done

# Deploy
git add . && git commit -m "nova pagina: [nome]" && git push
```
